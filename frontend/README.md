# Frontend

React 19 + Vite 6 frontend for the Bloglist application.

## Scripts

```bash
npm run dev      # Start dev server
npm run build   # Build for production
npm run test   # Run tests
npm run lint   # Lint code
```

## Development

```bash
npm run dev
```

Access at http://localhost:5173

## Build

```bash
npm run build
```

Output goes to `backend/dist` (served by Express in production).
